# Spinner

<Description>
  Spinners, similar to Progress Bars, are used when retrieving data or performing slow computations that are indeterminate, spinners communicate to the user that content is loading.
</Description>

## Size
Use the `size` property to specify a small, medium, or large Spinner.

## Colors
Use the `background` and `color` properties to change the colors of the Spinners channel and indicator.

## Labels
Text can be displayed with the Spinner with the `label` property. The text can be displayed below or beside the Spinner by setting the `inline` property to `true` or `false`.

  <Story name="Labels">

      <Spinner size="large" label="Inline" inline="true" />
      <Spinner size="large" label="Not Inline" color="#00A895" />
    </div>
  </Story>
</Preview>

## Integration Example
Follow the example below to integrate the Spinner component into your application:

```jsx

import Spinner from '@gcc/autobahn-components';

const App = (props) => {

  return (
    <Spinner />
  );
};

export default App;
```
